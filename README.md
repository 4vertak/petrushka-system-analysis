# Тестовое задание: Junior Системный аналитик

Репозиторий содержит решение тестового задания на позицию **Junior Системный аналитик** в компанию [Effective Mobile](https://effective-mobile.ru/vacancies/SA)


## Задание 1. Анализ требований

Проанализировать фрагмент ТЗ на функционал корзины:
1. [x] Найти и перечислить все логические противоречия и недочеты.
2. [x] Предложить исправленную редакцию требований (фрагмент ТЗ).
3. [ ] Составить список уточняющих вопросов для Product-менеджера или бизнес-заказчика.

## Задание 2. Проектирование API

Разработать верхнеуровневую архитекутуру системы push-уведомлений в микросервисном окружении:
1. [ ] Написать пример REST API запроса, который будет вызываться при переходе пользователя на данный экран.
2. [ ] Привести пример JSON-ответа, соответствующий макету и включающий ссылки на внешние ресурсы.

## Задание 3. Архитектура push-уведомлений

Разработать верхнеуровневую архитектуру системы push-уведомлений в микросервисном окружении:
1. [ ] Визуальная схема
2. [ ] Дать описание как должна работать отправка PUSH уведомлений в данном приложении. \
    *Гипотеза: речь скорее всего идет об описании потоков данных и компонентов?*
